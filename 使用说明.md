# 文件收纳箱 — 使用说明

> **EXE 文件位置**：`dist/文件收纳箱v6.6.exe`
>
> 打包完成后，exe 文件位于项目目录下的 `dist` 文件夹中。将整个 `dist` 文件夹复制到任意 Windows 电脑，双击 exe 即可直接运行，无需安装 Python 或其他任何依赖。

## 一、软件简介

**文件收纳箱** 是一款纯本地运行的桌面文件管理工具，帮你把分散在电脑各处的视频、图片、文档自动归类展示。全程只保存文件路径，**绝不复制、不移动、不修改你的原始文件**。

---

## 二、启动软件

双击 `文件收纳箱.exe` 即可启动，无需安装任何额外环境。

> 首次启动若提示需要 WebView2 Runtime，点击确认后系统会自动安装（Win10 1809+ / Win11 通常已自带）。

---

## 三、添加文件夹

### 方式一：拖拽添加（推荐）

将任意文件夹直接拖入软件窗口任意位置，松手即可自动收录。

> 支持 C/D/E/F 等任意磁盘。若拖拽偶尔不识别，请改用方式二。

### 方式二：选择目录

点击顶部 **"选择目录"** 按钮，在弹出的对话框中选择文件夹，确认后自动收录。

---

## 四、浏览文件

### 左侧导航栏

| 入口 | 说明 |
|------|------|
| **我的目录** | 左右双栏管理收录目录：左栏遍历收录（扫描内部文件），右栏项目收录（以文件夹为单位） |
| **视频** | 自动聚合所有目录中的视频文件 |
| **图片** | 自动聚合所有目录中的图片文件 |
| **文档** | 自动聚合所有目录中的文档文件 |
| **项目** | 以文件夹为单位管理项目，不遍历内部文件，支持收藏/删除/定位/批量操作 |
| **收藏 — 视频/图片/文档/项目** | 进入对应分类的收藏夹管理 |

### 顶部操作栏

- **搜索框**：输入关键词后按 **回车** 搜索；搜索框为空时按回车返回展示全部文件
- **排序**：名称 / 修改时间 / 创建时间 / **大小**，升序/降序一键切换
- **批量操作**：进入/退出批量选择模式，支持批量移除、永久删除、收藏
- **⚡ 释放内存**：清空前端媒体缓存和预览帧缓存，快速缓解视频区域内存占用
- **全部刷新**：重新扫描所有收录目录
- **🔄 局部刷新**：各页面右上角均带刷新按钮，仅重新渲染当前视图，修复操作后可能出现的 UI 状态不一致
- **视频预览开关**：视频视图排序栏带"预览"按钮，默认关闭，点击开启后悬停 1 秒触发 2.5 秒帧预览；关闭后视频卡片仍保留静态封面帧

### 分页栏

- **每页条数**：10 / 20 / 50 / 100 条/页可选（位于底部分页栏右侧）
- 分页导航居中显示，与内容保持适当间距

## 五、文件操作

### 打开文件

| 文件类型 | 操作方式 |
|----------|----------|
| 视频 | 点击卡片内置播放（mp4/webm/mov 等），其他格式自动调用系统播放器；鼠标悬停 1 秒触发 8 秒帧预览 |
| 图片 | 点击卡片查看大图，ESC 关闭，左右箭头键切换上一张/下一张 |
| 文档 | 点击卡片调用系统默认软件打开（Word/Excel/PDF 等） |

### 定位文件位置

点击文件卡片右下角的 **定位图标**，自动打开资源管理器并选中该文件。

### 收藏文件

点击文件卡片右上角的 **星形图标**（悬浮时显示）：
- 首次收藏会弹出窗口，可选择已有收藏夹或新建收藏夹
- 已收藏的文件显示黄色实心星
- 再次点击可取消收藏

---

## 六、收藏夹管理

### 进入收藏专区

在左侧导航栏点击 **"收藏 → 视频"** / **"收藏 → 图片"** / **"收藏 → 文档"** / **"收藏 → 项目"**，进入对应分类的收藏夹列表。

### 新建收藏夹

点击收藏夹列表页的 **"新建收藏夹"** 按钮，输入名称即可创建。

### 收藏夹操作

每个收藏夹卡片支持：
- **点击进入**：查看收藏夹内的文件
- **重命名**：修改收藏夹名称
- **搬迁**：批量剪切移动收藏夹内所有真实文件到统一文件夹（高危操作，二次确认）
- **删除**：删除收藏夹（仅删除软件记录，不删原文件）

### 收藏夹内文件预览

收藏夹内的文件预览完全复用原有逻辑：视频播放、图片大图、文档系统打开，左右箭头键跨文件切换均支持。

---

## 七、目录管理

在 **"我的目录"** 页面，采用左右双栏布局：

### 左栏 — 遍历收录

此处收录的文件夹会被递归扫描，内部的所有图片、视频、文档会自动归类到对应分类。

| 图标 | 功能 |
|------|------|
| 刷新 | 重新扫描该目录 |
| 定位 | 打开资源管理器并选中该文件夹 |
| 移除 | 从软件记录中移除该目录（**不删原文件**） |
| 永久删除 | 真实删除本地文件夹（**红色高危按钮，二次确认**） |

### 右栏 — 项目收录

此处收录的文件夹以项目为单位进行管理，**不会遍历内部文件**，直接出现在"项目"专区中。

| 图标 | 功能 |
|------|------|
| 定位 | 打开资源管理器并选中该文件夹 |
| 移除 | 从软件记录中移除该项目（**不删原文件**） |
| 永久删除 | 真实删除本地文件夹（**红色高危按钮，二次确认**） |

> **注意**："移除"只删除软件内的路径记录，磁盘上的文件夹完好无损；"永久删除"才是真正的文件删除，请谨慎操作。

## 八、快捷键

| 快捷键 | 功能 |
|--------|------|
| `ESC` | 关闭图片大图预览 / 视频播放模态框 |
| `←` / `→` | 图片/视频预览时切换上一个/下一个文件 |
| `Enter` | 确认弹窗中的操作 |

## 九、常见问题

**Q：拖拽添加失败，提示"未能识别拖拽路径"？**
A：这是 WebView2 的限制，某些情况下无法获取拖拽路径。请改用顶部 **"选择目录"** 按钮添加。

**Q：双击 mkv / rmvb 没反应？**
A：内置播放器不支持这些格式，软件会自动调用你电脑上的默认播放器（如 PotPlayer / VLC）。

**Q：大视频 (>500MB) 无法预览？**
A：内置播放器通过内存加载视频，超过 500MB 会限制，请点击卡片右下角的 **定位图标** 用系统播放器打开。

**Q：软件卸载后我的文件还在吗？**
A：在。软件全程只保存路径字符串，所有原始文件都在原来的位置。删除软件或清空 `config.json` 都不会动你的文件。

**Q：换了电脑怎么迁移数据？**
A：复制 `C:\Users\你的用户名\.file_aggregator\config.json` 到新电脑的同名位置即可。

**Q：收藏夹里的文件被移动/重命名了怎么办？**
A：收藏夹只保存路径，原文件被移动后路径会失效。建议整理好文件后重新收藏。

## 十、数据备份

所有配置保存在：

```
C:\Users\你的用户名\.file_aggregator\config.json
```

- 重装系统前可复制此文件备份
- 换电脑时复制此文件即可恢复全部收录记录和收藏夹
